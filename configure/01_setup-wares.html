Setup WARES on Workstations¶

Configure Workstations¶

Prepare each workstation to run WARES using workstation management guidelines. In addition, each workstation will require batch files and pif files as described at DOS Programs and NET USE and EMS Memory Under MS Windows. Revised versions of desktop icons and batch files may be obtained from the Download links page.

Once the WARES icon is working on the desktop, use these suggestions to improve the way the WARES icon works on Windows 7, 8, and 10.

  1. Double-click the WARES icon to start a WARES console session.
  2. Right-click the console icon at the left of the menu bar, and choose Properties.
  3. Check item Properties ‣ Options ‣ Use Legacy Console. (This will enable the mouse and function keys.)
  4. Set Properties ‣ Fonts to use Consolas size 28. Resize the console window by making the font size larger or smaller.
