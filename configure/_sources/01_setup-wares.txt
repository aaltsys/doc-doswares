#############################
Setup WARES on Workstations
#############################

Configure Workstations
=============================

Prepare each workstation to run WARES using 
`workstation management guidelines <http://servers.aaltsys.info/workstation>`_.
In addition, each workstation will require batch files and pif files as 
described at :ref:`netuse` and :ref:`dosems`. Revised versions of desktop icons 
and batch files may be obtained from the :ref:`downloads` page.

Once the :program:`WARES` icon is working on the desktop, use these suggestions 
to improve the way the :program:`WARES` icon works on Windows 7, 8, and 10.

#. Double-click the :program:`WARES` icon to start a WARES console session.
#. Right-click the console icon at the left of the menu bar, and choose
   :menuselection:`Properties`.
#. Check item :menuselection:`Properties --> Options --> Use Legacy Console`.
   (This will enable the mouse and function keys.)
#. Set :menuselection:`Properties --> Fonts` to use **Consolas** size **28**.
   Resize the console window by making the font size larger or smaller.
